# 🏦 Secure Banking System

A Python-based Secure Banking System that allows users to create accounts, authenticate securely, manage balances, and track transactions.

## 🚀 Features

* User Account Creation
* Secure Login Authentication
* SHA-256 Password Hashing
* Password Strength Validation
* Deposit Money
* Withdraw Money
* Balance Inquiry
* Transaction History
* JSON-based Data Storage
* Modular Project Structure

## 🛠️ Technologies Used

* Python
* JSON
* Hashlib
* Regular Expressions (Regex)

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/aastha-ui/Secure-Banking-System.git
```

2. Navigate to the project folder

```bash
cd Secure-Banking-System
```

3. Run the application

```bash
python main.py
```
